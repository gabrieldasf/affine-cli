# AFFiNE Printing Press Inputs

- `schema.gql`: copy of the SDL generated in `qtz-affine`.
- `affine-api-inventory.json`: inventory of the schema and client GraphQL documents.
- `affine-graphql.yaml`: spec interna consumida pelo Printing Press.

The spec exposes `graphql execute` for any GraphQL document and named commands for the existing `.gql` operations.
